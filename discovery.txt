.cursor/rules/product-rules.md
# Product Rules

- This product is a decision engine, not a generic job board.
- The app's purpose is to identify high-fit founding/builder GTM roles and help the user act quickly.
- A role is more important than its title; signals in the description, company stage, and ownership matter.
- The product must always preserve two modes:
  - Cinematic radar mode
  - Tactical productivity mode
- Tactical usability beats flashy visuals when there is a conflict.
- Public-facing content is secondary to the private operating system in v1.
- Every surfaced role should support clear action states:
  new, reviewed, shortlisted, applied, outreach sent, interviewing, archived.
.cursor/rules/design-rules.md
# Design Rules

- The visual language is sonar/radar command center, not hacker-movie gimmick.
- Every visual effect must encode meaning.
- Use restrained motion, not decorative motion.
- Build the tactical mode before polishing cinematic mode.
- Sound must be easy to mute and never required for usability.
- Primary interface principles:
  - dark premium command console
  - phosphor/radar energy
  - strong hierarchy
  - readable typography
  - meaningful glow
  - glass-like panels only if readability remains high
- Maintain consistency with docs/FRONTEND_GUIDELINES.md and docs/VISUAL_LANGUAGE_MAP.md.
- Any new visual inspiration must be logged in docs/INSPIRATION_LOG.md.

# Engineering Rules

- Prefer boring, stable architecture over cleverness.
- Start with the data model, ingestion path, normalization, deduplication, and scoring before advanced UI.
- Prefer small, composable components.
- Prefer pure functions for scoring and normalization logic.
- Prefer explicit types and interfaces.
- Prefer testable logic for nontrivial business rules.
- No silent new packages.
- No silent new external services.
- No schema changes without updating docs/BACKEND_STRUCTURE.md.
- No API contract changes without updating docs/DATA_FLOW.md and relevant docs.
- Add instrumentation/logging for ingestion jobs and scoring runs.
