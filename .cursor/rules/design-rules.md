# Design Rules

- The visual language is sonar/radar command center, not hacker-movie gimmick.
- Every visual effect must encode meaning.
- Use restrained motion, not decorative motion.
- Build the tactical mode before polishing cinematic mode.
- Sound must be easy to mute and never required for usability.
- Primary interface principles:
  - dark premium command console
  - phosphor/radar energy
  - strong hierarchy
  - readable typography
  - meaningful glow
  - glass-like panels only if readability remains high
- Maintain consistency with docs/FRONTEND_GUIDELINES.md and docs/VISUAL_LANGUAGE_MAP.md.
- Any new visual inspiration must be logged in docs/INSPIRATION_LOG.md.
